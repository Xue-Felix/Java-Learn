# 数据类型

## 包装类型

- boolean/1
- byte/8
- char/16
- short/16
- int/32
- float/32
- long/64
- double/64

> 每个类型都有对应的包装类型，基本类型与其对应的包装类型之间的赋值使用自动装箱与拆箱完成

```java
Integer x = 2;  // 装箱
int y = x;      // 拆箱
```

## 缓存池

- new Integer(123)每次都会新建一个对象
- Integer.valueOf(123)会使用缓存池中的对象，对此
